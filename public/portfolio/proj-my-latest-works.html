Blog post
Let's talk Web Accessibility

Let's talk Web Accessibility
The simple answer for Web Accessibility is that all internet content should be built in a way that allows everybody to physically and mentally access and use it. The simple reason for is that a lot of the web content is vital and critical to maintaining people's everyday lives! In today's world, the internet is a fountain of life-sustaining information that everybody drinks from in order to survive.
